[cite_start]

A powerful graph-based OSINT tool for mapping relationships between people, organizations, and infrastructure[cite: 8]. Domain & Infrastructure Reconnaissance

[cite_start]

Mapping a target's external digital footprint by discovering domains, subdomains, IPs, and exposed services is a foundational first step in security engagements[cite: 14, 180].

theHarvester

[cite_start]

A classic command-line tool for passive reconnaissance, gathering emails, subdomains, and employee names from over 40 public sources like search engines and breach datasets[cite: 15, 16, 187]. [cite_start]It aggregates a wide spectrum of digital footprint data into a single report[cite: 103].

[cite_start]
Recon Type: Passive [cite: 240][cite_start].
OPSEC: Queries to public sources originate from the operator's IP[cite: 192, 240]. [cite_start]Must be run from a non-attributable network for covert engagements[cite: 193].

Nmap

[cite_start]

The global standard for active network discovery and security auditing[cite: 195]. [cite_start]It sends packets to discover open ports, running services, and OS versions on target hosts[cite: 59, 196]. [cite_start]Its power is extended by the Nmap Scripting Engine (NSE) which can check for specific vulnerabilities[cite: 60, 197].

[cite_start]
Recon Type: Active [cite: 240][cite_start].
OPSEC: Inherently overt[cite: 208]. [cite_start]Generates significant, detectable network traffic[cite: 208, 240]. [cite_start]Use must be authorized [cite: 210, 240][cite_start].
License: NPSL, which prohibits redistribution in proprietary products without an OEM license[cite: 201, 203].

[cite_start]

Investigations involving Tor hidden services require specialized tools and a disciplined methodology, starting with safe discovery and moving to targeted analysis[cite: 304, 305].

[cite_start]

A reputable public search engine for Tor hidden services, accessible from the clearnet or a .onion address[cite: 38, 308, 309]. [cite_start]It serves as a relatively safe entry point for dark web discovery and actively filters abusive content[cite: 310, 312, 314].

[cite_start]
Phase: Discovery [cite: 359][cite_start].
OPSEC: Mandatory to access via its .onion address from the Tor Browser for operational use[cite: 317]. [cite_start]Search queries are logged by the provider[cite: 318, 359].
